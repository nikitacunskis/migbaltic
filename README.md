## About project
Requirment:

PHP: 8.1.4 

MariaDB: 10.4.24-MariaDB

Comoser: latest

NPM: latest

Git: latest

## Installation

Create database and set-up .env file.

`git clone https://github.com/nikitacunskis/ict.git`

`composer install`

`npm install`

`php artisan serve`

`nmp run dev`

`php artisan migrate`
